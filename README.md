# Animon: Resonance

Um RPG de captura de criaturas 2D em estilo retro, criado totalmente em HTML5, CSS e JavaScript Vanilla, sem a necessidade de assets externos.

## Como Jogar

1. **Abra o arquivo `index.html`** no seu navegador web preferido (Google Chrome, Firefox, Edge, Safari). Não é necessário rodar um servidor local, pois o jogo não faz requisições externas para assets (as artes e sons são gerados proceduralmente!).
2. Na tela de título, digite seu nome, escolha a cor da sua jaqueta e clique em **Iniciar Jornada** (ou pressione ENTER).
3. **Controles:**
   - **W, A, S, D** ou **Setas do Teclado**: Mover o personagem.
   - **Z** ou **Enter**: Interagir com NPCs, ler placas e avançar diálogos.
   - **Esc**: Abrir o Menu de Pausa (Equipe, Mochila, Salvar).
4. Fale com o Professor Hawthorne (NPC laranja) logo na entrada para ganhar seu primeiro Animon (Leaflora).
5. Ande na grama alta (blocos mais escuros) para encontrar Animons selvagens.
6. Na batalha, escolha **Lutar** para atacar ou **Mochila** para usar *Cápsulas Animon* e capturar novas criaturas.
7. Derrote o Rival e explore até o Templo Verde para desafiar a Líder Flora!

## Tecnologias Usadas
- **Canvas HTML5** para renderização a 60 FPS do mapa em formato de grade (Tilemap) e desenhos procedurais.
- **Web Audio API** para síntese de efeitos sonoros 8-bit (sem precisar de arquivos `.mp3` ou `.wav`).
- **localStorage** para o sistema de salvamento do jogo.

Divirta-se capturando e evoluindo seus Animons em Aethelgard!
